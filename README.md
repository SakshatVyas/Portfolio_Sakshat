# Sakshat Vyas - 3D Developer Portfolio

A modern, highly interactive 3D developer portfolio built with React, Three.js, TailwindCSS, and Framer Motion — designed to showcase my projects, skills, and experience with clean, responsive, and engaging UI.

## ✨ Features

- **3D Interactive Visuals:** Smooth 3D rendering powered by **React Three Fiber** and **Drei**.
- **Scroll Animations:** Fluid transitions and scroll-based interactions using **Framer Motion**.
- **Fully Responsive Design:** Seamless experience across mobile, tablet, and desktop using **TailwindCSS**.
- **Modern Performance:** Lightning-fast development and build times with **Vite**.

---

## 🛠 Tech Stack

- **Frontend & UI:** React, Vite, Tailwind CSS, HTML5, CSS3
- **3D & Animation:** React Three Fiber, Drei, Three.js, Framer Motion
- **Languages:** JavaScript, Java, Python
- **Database:** SQL (MySQL)

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/SakshatVyas/Portfolio.git
cd Portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```
The app will be available at [http://localhost:5173](http://localhost:5173).
